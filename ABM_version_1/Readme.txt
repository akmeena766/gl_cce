This is the 1st version of the Adaptive Boundary Method (ABM).

In this version:
(1) individual_image: This folder contains the programs for simulating the light curve for constant macrolens shear and convergence.
(2) caustic_crossing: This folder contains the programs for simulating the light curve for caustic crossing events. 
 
In both folders, one can change the simulation parameters in 'input.py' file.
